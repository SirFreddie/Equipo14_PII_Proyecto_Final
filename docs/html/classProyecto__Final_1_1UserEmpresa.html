Detailed Description

Esta clase representa al usuario de la Empresa. La función de esta clase es la de representar a un usuario que interactúa con el sistema que se identifica como empresa. Debido a esto, la única responsabilidad de esta clase es la de proveer con un nexo entre las interacciones de usuario y los datos de este usuario, los cuales se almacenan en la clase "Empresa" y los accede mediante el patrón de Delegación. Por lo cual también sigue con el patrón SRP.
